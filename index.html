Speaker Notes — Slide 2

So let's start with what the research actually says. [pause] METR ran a randomized controlled trial last year. Sixteen developers, two hundred forty-six tasks, real open-source repos. The result? Developers were nineteen percent slower with AI. But here's what's interesting—they believed they were twenty percent faster. There's a gap between perception and reality, and nobody's talking about it. [pause] Faros AI found something similar at scale. Yes, more tasks got done. More PRs shipped. But review time nearly doubled. Net throughput? Flat. [pause] DORA—Google's annual report, thirty-nine thousand developers—found a slight decrease in both throughput and stability with AI adoption. [pause] I'm not saying these tools don't work. I'm saying we don't understand why they sometimes don't. And that's a fundamentally different kind of problem.

Architecture

The Stateless Mind

Every AI coding tool works the same way underneath. The model has no memory.

Claude Code, Cursor, Copilot, Codex, Gemini CLI—under the hood, they all work the same way. Every single turn, the full conversation gets re-sent to the API. Every file your agent reads, every command output, every grep result. It piles up, and it all ships back on every turn that follows.

Speaker Notes — Slide 3

Here's what I think is actually happening underneath all those numbers. [pause] Every AI coding tool you use—Claude Code, Cursor, Copilot, Codex—works the same way. The model has no memory. None. Every single turn, the entire conversation gets packed up and re-sent to the API. Every file it read. Every command output. All of it. Shipped back, every turn. [pause] Run one GitHub command. Fifty-nine kilobytes come back. Fifteen thousand tokens. Next turn, those tokens ship again. And again. Fifty turns in, that single command has cost you seven hundred fifty thousand input tokens. [pause] Eventually the window fills up. The agent compacts—it summarizes the conversation and throws away the originals. And just like that, it forgets what it read, what it decided, what it was in the middle of building. You start over. You re-explain your codebase to a model that just lost its mind. [pause] Research shows a fifty percent accuracy drop after thirty-two thousand tokens. Your agent literally gets worse at its job the longer it works. That's not a feature. That's architectural amnesia.
